# 问题描述

在对话收件箱列表中，部分会话标题过长（如来自 Dify 会话名或长用户问题摘要），未做行数限制，**标题占满多行**，列表项高度过大，影响预览与扫视体验。

**复现条件**：存在 `title` 或展示名较长的会话行；进入 `pages/chat/index` 即可见。

# 根因分析

列表模板中标题使用 `<text class="name">` 未设置 `overflow` / `-webkit-line-clamp`，长文本在 flex 行内自动换行且无上限，导致列表项纵向过高。

- `hipp-or/hipp-uni/pages/chat/index.vue`：标题区增加可收缩容器与两行截断样式；标签与标题分行布局调整。
